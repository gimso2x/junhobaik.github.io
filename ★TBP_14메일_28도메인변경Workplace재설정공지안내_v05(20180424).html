안녕하세요? SK telecom API Portal 입니다.

항상 SK telecom API Portal를 이용해주시는 고객님들께 감사 드립니다.
안정적인 서비스 운영을 위하여 하기 중요 공지사항을 한 번 더 전달해드립니다.

2017년 12월 27일 공지 드린 바와 같이, 도메인 변경 작업으로 인하여 DNS 업체에 요청하시어 메일의 MX
레코드 변경 및 SPF 레코드 등록을 진행하여 주시기 바랍니다.
 
 
  • API Portal 도메인 MX 레코드 값 : gwspam.sktelecomapi.com. / 호스트는 @
  • API Portal 도메인 SPF 레코드 값 : "v=ip4:175.126.131.57 ip4:175.126.131.58
    ip4:175.126.131.68 ip4:175.126.131.69 ~all"
 
 
※  도메인이 변경됨에 따라 MX레코드를 변경하지 않을 경우, 메일 수신 서비스가 불가할 수 있습니다.
(MX레코드가 변경되는 동안 일시적으로 메일 수신이 불가할 수 있습니다.)
SPF 레코드 값이 등록되지 않은 경우, Daum(다음)으로의 메일 발송이 원활하지 않을 수 있습니다.
이에 따라, 번거로우시더라도 위의 내용을 재설정하시어 서비스를 이용해 주시기 바랍니다.
 
 
궁금하신 점이나 보다 자세한 작업 관련 안내가 필요하신 경우 SK telecom API Portal 고객센터(1600-0990)로
문의 주시기 바랍니다.

감사합니다.
